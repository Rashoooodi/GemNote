## Features

- AI-ready home timeline that showcases recent summaries, Gemini model usage, and quick actions to capture or configure notes.
- Recording workspace with toggles for Gemini-powered summaries and tag suggestions so you can simulate automations before wiring the backend.
- Detailed note screen highlighting summaries, suggested tags, and attachment slots that illustrate the full note lifecycle.
- Persistent Gemini settings stored locally with DataStore, including secure API key storage and selectable models.
- Room-backed data layer scaffolding for offline note storage alongside Retrofit + OkHttp clients for future syncs.
- Material 3 expressive theme with automatic dark mode and dynamic color support on Android 12+.
- Zero Firebase dependencies are required to build; Google Services is only applied when you provide configuration files.

> ℹ️ Email support is not currently implemented in the application.

## Getting started

Open in Android Studio, let it sync. If wrapper missing, run: `gradle wrapper`
